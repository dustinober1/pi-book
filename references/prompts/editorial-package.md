# Thriller Packaging Prompt

Use the `thriller-forge-for-pi` skill.

## Goal

Prepare the active thriller book and series package for publication or sales-page testing.

## Read first

- `series/series-bible.md`
- `series/series-arc.md`
- `series/market-package.md`
- active book `book-bible.md`
- active book `package.md`
- active book `audits/manuscript-audit.md`
- active book `revision-tickets.md`

## Required outputs

Update only:

- `series/market-package.md`
- active book `package.md`
- active book `revision-tickets.md`, if packaging exposes a concrete problem

## Package sections

Include:

- title options
- series line
- short hook
- full blurb
- category and keyword notes
- cover concept
- first-page promise
- read-through hook for the next book
- open risks before publication

Do not claim validation that has not happened. Mark untested assumptions clearly.
